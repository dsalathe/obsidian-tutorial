
# Shortcuts
`Ctrl + Shift + P`: create a new `People`
`Ctrl + Shift + D`: create a new delivery (project, PoC...)
`Ctrl + Shift + M`: create a new meeting

`Ctrl + O`: Search for a specific note
`Ctrl + N`: Create a new note
